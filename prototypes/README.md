# Prototypes & Experimental Scripts

This directory contains **experimental and prototype code** used during development and testing. These scripts are **not intended for production use** and may not be maintained.

- Quick proof-of-concept scripts
- Manual testing and verification tools
- Platform-specific experiments
- Throwaway code for exploring ideas
